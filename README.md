# 🍎React Blog Study

코딩애플 React 강좌 실습 프로젝트입니다.

## 📋실습 내용

- useState를 활용한 상태 관리
- 컴포넌트 분리 및 props 전달
- 배열 상태 불변성 유지 (얕은 복사)
- 조건부 렌더링 (삼항연산자)
- 이벤트 버블링 방지 (stopPropagation)

## 🔍주요 기능

| 기능 | 설명 |
|------|------|
| 게시글 목록 | 제목, 날짜 렌더링 |
| 좋아요 | 게시글별 카운트 누적 |
| 게시글 삭제 | 선택한 항목 제거 |
| 모달 | 클릭 시 상세보기 + 제목 수정 |
| 가나다 정렬 | 오름차순 정렬 |
| 게시글 등록 | 입력 후 목록에 추가 |

## ⚙️기술 스택

- React 19
- Vite
- JavaScript (JSX)

## 🚀실행 방법

```bash
npm install
npm run dev
```
