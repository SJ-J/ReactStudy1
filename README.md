# 🍎React Blog Study

코딩애플 React 강좌 실습 프로젝트입니다.

## 📋실습 내용

- useState를 활용한 상태 관리
- 컴포넌트 분리 및 props 전달
- 배열 상태 불변성 유지 (얕은 복사)
- 조건부 렌더링 (삼항연산자)
- 이벤트 버블링 방지 (stopPropagation)
- 여러 배열 state 간 동기화 관리 (postTitle, postDate, goodCnt)

## 🔍주요 기능

| 기능 | 설명 |
|------|------|
| 게시글 목록 | 제목, 날짜 렌더링 |
| 좋아요 | 게시글별 카운트 누적 |
| 게시글 삭제 | 선택한 항목 제거 및 관련 state 동기화 |
| 모달 | 클릭 시 상세보기 + 제목 수정 |
| 가나다 정렬 | 오름차순 정렬 |
| 게시글 등록 | 입력 후 목록 추가 + 현재 날짜 자동 적용 |
| 제목 즉시 변경 | 버튼 클릭 시 선택된 게시글 제목 일괄 변경 |
| 이미지 토글 | 버튼 클릭 시 이미지 영역 표시/숨김 |

## 🧩컴포넌트 구조

| 컴포넌트 | 설명 |
|---------|------|
| `App` | 메인 컴포넌트, 전체 state 관리 |
| `Modal` | 게시글 상세보기 및 제목 수정 모달 |
| `SmileCovy` | 웃는 코비 이미지 토글 컴포넌트 |
| `WaitCovy` | 의젓한 코비 이미지 토글 컴포넌트 |

## ⚙️기술 스택

- React 19
- Vite
- JavaScript (JSX)
